Manual de Uso

Controles de Pontuação

  • T (Tórax): Clique para adicionar 2 pontos (chute).
  • C (Cabeça): Clique para adicionar 3 pontos (chute).
  • S (Soco): Clique para adicionar 1 ponto (soco no tórax).
  • B (Bônus): Ativa o bônus de giro. O próximo chute (T ou C) valerá +2 pontos.
  • G (Gam-geon/Falta): Adiciona 1 ponto para o oponente.

Correção de Erros

  • Clique Longo: Para remover pontos ou uma falta, pressione e segure o botão correspondente.
  • Remover com Bônus: Para remover um chute com giro, ative o botão 'B' e então pressione e segure 'T' ou 'C'.
  • Ajuste Manual: Com o tempo pausado, clique em "Ajuste Manual" para exibir botões de '+' e '-' e corrigir o placar.

Controles Centrais

  • Cronômetro: Clique no tempo antes do início do 1º round para ajustar sua duração.
  • Botão Play/Pause: Inicia/para o tempo. Após o intervalo, um clique inicia o próximo round.
  • Botão Recomeçar: Aparece com o tempo pausado e permite reiniciar toda a partida.
  • Intervalo de Descanso: Um timer de 30s é iniciado automaticamente entre os rounds no lugar do texto "ROUND".
