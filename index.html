Manual de Uso

Controles de Pontuação

  • T (Tórax): Clique para adicionar 2 pontos.
  • C (Cabeça): Clique para adicionar 3 pontos.
  • B (Bônus): Clique para ativar o bônus de giro. O botão ficará amarelo. O próximo golpe (T ou C) valerá +2 pontos.
  • G (Gam-geon/Falta): Clique para aplicar uma falta. Isso adiciona 1 ponto para o oponente.

Correção de Erros (Clique Longo)

Para remover uma pontuação ou falta, pressione e segure o botão correspondente por 1 segundo.

  • Segurar T ou C: Remove os pontos do golpe. Se o golpe foi com bônus (giro), remove a pontuação completa.
  • Segurar G: Remove a última falta e retira o ponto que foi dado ao oponente.

Controles Centrais

  • Cronômetro (ex: 01:30): Clique diretamente sobre o tempo para ajustar a duração do round.
  • Botão Play/Pause: Inicia e para o tempo. No final de um round, clique para avançar para o próximo.

Fim de Luta e Rounds

  • Um atleta vence a luta ao ganhar 2 rounds.
  • A luta pode terminar antes se uma das condições abaixo for atingida:
    • 5 Faltas: Ao acumular 5 faltas, o atleta perde o round.
    • Diferença de 20 Pontos: Uma diferença de 20 pontos a partir do 2º round resulta em vitória imediata.
